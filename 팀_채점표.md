## 1000회 시뮬레이션 요약 (seed=42)

| 통계 | 값 |
|:--|:--|
| 지표 점수 범위 | **6 ~ 46점** |

### RFP 미달 빈도

| 항목 | 미달 건수 | 비율 |
|:--|--:|--:|
| 오픈스페이스 | 174 | 17.4% |
| 저렴한 집 | 77 | 7.7% |
| 일자리 | 68 | 6.8% |
| 거주성 | 40 | 4.0% |
| 이윤 | 12 | 1.2% |
| 탄소 | 3 | 0.3% |

### 지표 점수 분포 (주요 구간)

| 점수 | 건수 | 비율 |
|:--|--:|--:|
| 36~40 | 452 | 45.2% |
| 30~35 | 285 | 28.5% |
| 41~46 | 282 | 28.2% |
| 29 이하 | 81 | 8.1% |

### 해석

- 랜덤 1000건 중 **40점 이상** ≈ 28%, **36점 이상** ≈ 73%
- 1등급 후보: 지표 **40점+** + RFP 전항목 충족 + 발표 가산
- **오픈스페이스 10%** 미달이 가장 흔한 RFP 감점 원인

## 현장 채점 양식

| 팀 | 일자리 /10 | 이윤 /10 | 탄소 /10 | 거주성 /10 | 저렴한집 /10 | 구간합 /50 | RFP감 | **지표 /50** | 발표+ /10 | **합계 /60** | 순위 |
|:--|--:|--:|--:|--:|--:|--:|--:|--:|--:|--:|--:|
| | | | | | | | | | | | |

**RFP 미달 (해당 시 −3)**: □일자리 □저렴한집 □오픈 □탄소 □거주성 □이윤

---

## 계산식

```
구간합 = 일자리점 + 이윤점 + 탄소점 + 거주성점 + 저렴한집점  (최대 50)
지표점 = max(0, 구간합 − RFP미달건수×3)                    (최대 50)
합계   = 지표점 + 발표가산                                  (최대 60)
```
